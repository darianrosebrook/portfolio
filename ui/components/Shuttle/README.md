# Shuttle (Compound)

Scaffolded component following system standards.

## Usage

```tsx
import Shuttle from '@/ui/components/Shuttle';

<Shuttle>{/* content */}</Shuttle>;

// Tokens live in Shuttle.tokens.json and are bootstrapped in Shuttle.tokens.generated.scss
```
