# Avatar (Compound)

Scaffolded component following system standards.

## Usage

```tsx
import Avatar from '@/ui/components/Avatar';

<Avatar>{/* content */}</Avatar>;

// Tokens live in Avatar.tokens.json and are bootstrapped in Avatar.tokens.generated.scss
```
